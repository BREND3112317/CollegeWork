29. 以 MARIE 組合語言寫出下列的碼片段:

```
if X > 1 then
    Y = X + X;
    X = 0;
endif;
Y = Y + 1;
```
  • MARIE
    IF, 	LOAD X 
        SUBT ONE 
        SKIPCOND 800 
        JUMP ENDIF 
THEN, 	LOAD X 
        ADD X 
        STORE Y 
        LOAD ZERO 
        STORE X 
ENDIF,  LOAD Y 
        ADD ONE 
        STORE Y 
        HALT
X,	    Dec 0
ONE,	Dec 1
ZERO,	Dec 1
