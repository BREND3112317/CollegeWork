### 29. 以 MARIE 組合語言寫出下列的碼片段:
    ```
    if X > 1 then
        Y = X + X;
        X = 0;
    endif;
    Y = Y + 1;
    ```
- MARIE
    ```	
    IF, 	LOAD X 
            SUBT ONE 
            SKIPCOND 800 
            JUMP ENDIF 
    THEN, 	LOAD X 
            ADD X 
            STORE Y 
            LOAD ZERO 
            STORE X 
    ENDIF,  LOAD Y 
            ADD ONE 
            STORE Y 
            HALT
    X,	    Dec 0
    ONE,	Dec 1
    ZERO,	Dec 1
    ```

### 34. 寫出下列碼片段的 MARIE 組合語言程式。(提示: 將 for 迴圈改成 while 迴圈。)

